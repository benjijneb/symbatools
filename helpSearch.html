Cet outil offre deux fonctionnalités :
  • Rechercher des talents/pouvoirs mystiques/rituels/traits/atouts/fardeaux et les afficher sous forme de cartes
  • Construire rapidement une fiche de PC/NPC en saisissant les caractéristiques et en épinglant les cartes désirées

La recherche

Pour rechercher des élements, le champ de recherche fonctionne par mots clés séparés par des virgules, en mode "contient". Une virgule à la fin réaffiche l'ensemble des résultats. La recherche n'est réalisée que sur le nom des cartes.
Exemples
  • "berserk" ou même "erserk" affichera le talent Berserker
  • "berserk," ou même "erserk," affichera tous les résultats (virgule à la fin)
  • "berserk,acrob" ou "berserk, acrob" affichera les talents Berserker et Acrobatie
Les filtres
En plus de la barre de recherche, vous pouvez sélectionner des filtres pour limiter les résutats. Les filtres se cumulent entre eux ainsi qu'avec les mots clés de la barre de recherche.
Afficher les descriptions
La case à cocher "Afficher les descriptions" permet d'afficher la description dans les cartes car celle-ci est cachée par défaut.
Compact
La case à cocher "Compact" permet de cacher la partie construction fiche PC/NPC et de n'afficher que la partie recherche.
Sélecteur de règles
Le sélecteur "Règles" permet de changer les textes affichés dans les différentes cartes (talents/pouvoirs mystiques/rituels/traits/atouts/fardeaux), en basculant entre les règles issues des livres de base et le travail d'équilibrage réalisé par Marco Pagnini : Symbaroum (Final)

Construction rapide fiche PC/NPC

Les caractéristiques à saisir correspondent à celles présentes sur une fiche de personnage Symbaroum.

  • {{'CATEG' | translate}} : Catégorie du PC/NPC (ex. "elfe d'hiver", "templier de prios", "dieu sanglier", etc.)
  • {{'AGI' | translate}} : {{'ATTR_AGI' | translate}}
  • {{'FOR' | translate}} : {{'ATTR_FOR' | translate}}
  • {{'PRE' | translate}} : {{'ATTR_PRE' | translate}}
  • {{'VOL' | translate}} : {{'ATTR_VOL' | translate}}
  • {{'VIG' | translate}} : {{'ATTR_VIG' | translate}}
  • {{'DIS' | translate}} : {{'ATTR_DIS' | translate}}
  • {{'AST' | translate}} : {{'ATTR_AST' | translate}}
  • {{'PER' | translate}} : {{'ATTR_PER' | translate}}
  • {{'INIT' | translate}} : {{'INITIATIVE' | translate}}
  • {{'DEF' | translate}} : {{'DEFENSE' | translate}}
  • {{'END' | translate}} : {{'ENDURANCE' | translate}}
  • {{'SD' | translate}} : {{'RES_DOUL' | translate}}
  • {{'SC' | translate}} : {{'SEUIL_CORR' | translate}}
  • {{'CP' | translate}} : {{'PERM_CORR' | translate}}

Selection des cartes
Les talents, pouvoirs mystiques, traits, atouts et fardeaux peuvent être épignés. A quoi cela sert-il ? Cela permet de sélectionner les différents talents et autres éléments que possède le PC/NPC pour l'export/import et d'utiliser la case "Epinglés seulement".
Les cartes peuvent être épinglées de plusieurs manières :
  • En cochant la case globale à côté du nom de la carte
  • En sélectionnant un ou plusieurs niveaux dans la carte
Epinglés seulement
La case à cocher Epinglés seulement permet de n'afficher que ce qui a été épinglé de manière à n'afficher que les éléments possédés par un PC/NPC.
Import/Export des caractéristiques
Une fois les caractéristiques saisies et les cartes sélectionnées, il est possible de réaliser plusieurs actions :
  • Bouton {{'SAVE' | translate}} : Sauvegarder la fiche sous plusieurs formats.
  • Bouton {{'IMPORT_JSON' | translate}} : Réimporer une fiche sauvegardée sous forme d'un fichier JSON.
  • Bouton Stat Block : Afficher les caractéristiques saisies sous forme d'un Stat Block (version Starter Kit).
Sauvegardes
  • {{'B84_PAN_TITLE' | translate }} : Génère une URL permettant d'être ajoutée comme favori. L'URL générée contient les valeurs des champs de caractéristiques, encodées, celles-ci étant ensuite restaurées lors de l'accès à l'URL.
  • {{'EXPORT' | translate}} : Exporte les caractéristiques saisies sous forme d'un fichier JSON (télécharge le fichier depuis le navigateur), qui peut ensuite être réimporté. Toutes les valeurs des champs de caractéristiques sont sauvegardés ainsi que les épingés des cartes et la langue utilisé.
  • (Anglais seulement) {{'FOUNDRY_EXPORT' | translate }} : Copie les caractéristiques saisies sous forme d'un JSON dans le presse-papier. Ces informations peuvent ensuite être collées dans le prompt de la macro d'import de PC/NPC pour Symbaroum.fr du système Symbaroum de FoundryVTT.

This search tool offers two features :
  • Search abilities/mystical powers/traits/boons/burdens et display them as cards
  • Quickly build a PC/PNC sheet by filling the characteristics and pinning desired cards

Search

To search cards, the search field works with coma separated keywords, in "contains" mode. An ending coma displays again everything. The search is only made on cards names.
Examples
  • "berserk" or even "erserk" will display Berserker ability
  • "berserk," or even "erserk," will display all results (ending coma)
  • "berserk,acrob" or ""berserk, acrob" will display Berserker and Acrobat
Filters
In addition to the search field, you can select filters to limit the results. Filter are cumulatives between them and with the search field content.
Display descriptions
The "Display descriptions" checkbow allows to display descriptions in cards, they are hidden by default.
Compact
"Compact" checkbox allows to hide the build PC/NPC sheet part and to only display the search part of the interface.
Rules selector
The "Rules" selector allows to change the cards texts, switching from standard core rules to the balancing work of Marco Pagnini : Symbaroum (Final)

Cards selection
Abilities, mystical powers, traits, boons and burdens can be pinned as cards. Why ? Il allows you to select the cards owned by the PC/NPC and use them with the import/export feature, and with the "Pinned only" checkbox.
The cards can be pinned in several ways:
  • By checking the global checkbox next to the card name
  • By selecting one or more levels in the card
Pinned only
The Pinned Only checkbox allows you to display only what has been pinned so that only items owned by a PC/NPC are displayed.
Character sheet import/export
Once the characteristics have been entered and the cards selected, it is possible to perform several actions:
  • Button - {{'SAVE' | translate}} : Save character sheet under multiple formats.
  • Button - {{'IMPORT_JSON' | translate}} : Import a previously saved character sheet (JSON format).
  • Button - Stat Block : Display all entered characteristics as a Stat Block (Starter Kit version).
Saves
  • {{'B84_PAN_TITLE' | translate }} : Generates a URL to be added as a favorite. The generated URL contains the values of all the characteristics fields, encoded, which are then restored when the URL is accessed.
  • {{'EXPORT' | translate}} : Exports the entered characteristics as a JSON file (download the file from the browser), which can then be reimported. All characteristics field values are saved as well as the pinned cards and language used.
  • (English Only) {{'FOUNDRY_EXPORT' | translate }} : Copies the entered characteristics as a JSON to the clipboard. This information can then be pasted into the PC/NPC import for Symbaroum.fr macro prompt from the FoundryVTT Symbaroum system.
